Here is Dejavu Moe's personal git server.

I host my personal projects here and you can clone the source code of any project, if you want to commit patches or changes here, feel free to send me an e-mail.

Service composition:

  • Debian GNU/Linux - Lovely Linux distribution
  • cgit - A hyperfast web frontend for git repositories written in C.
  • Nginx - The web server
  • Cloudflare - Secure and fast DNS resolution and free global CDN acceleration

If you'd like to host a similar Git server on your machine, there's a post on my blog that describes the details.
